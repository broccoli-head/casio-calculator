# About
- Simple calculator app that is looking and behaving exactly the same as the original physical Casio model.
- As the additional feature, it's converting an input sequence into RPN (Reversed Polish notation) and then calculate it.
<br>Example: `2 + 3 * 4` will equal `14` instead of `20`
